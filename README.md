# rspp: Rock-Scissors-DoublePaper analysis code
Bart Massey

Rock-Scissors-DoublePaper (RSPP) is just
Rock-Scissors-Paper, but with a win by paper over rock
counting double.

This repo contains simulation code to analyze RSPP
strategies.
